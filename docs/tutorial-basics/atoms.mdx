# Atoms

## Atom

An atom is a constant whose value is the same as it's name. Defining an atom is same as a variable where an atom is prefixed with a `:`

Eg. :atom, :hello, :dex

```
> :atom
:atom
```

## Boolean

Boolean values represent the truth and false values. These are denoted by the keywords `true` and `false`

## Null

Null is the placeholder value that represents nothing. The null is denoted by `null` keyword
